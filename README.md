#Additional Checkout Attributes

WooCommerce checkout field editor allows you to add custom fields to the checkout page that gathers additional information from the potential customers. With the help 10 different types of fields, you can allow customers to provide you relevant information that is useful in delivering them the products or services. You can set different options with the additional fields like text area, radio button, or multi-select box, etc.

#Prerequisites

Wordpress must be installed, atleast 4.2.0.

WooCommerce must be installed.

#Installation

1. Go to Plugins -> Add New.
2. Click "Upload" right underneath "Install Plugins".
3. Upload the zip file that this readme was contained in.
4. Activate the plugin.
5. Go to Checkout Fields Attributes
